Agent 没有真正的工具调用 — tool_node.py 返回的是假数据（天气写死的），没有接入真实 API 或搜索
Agent 没有多轮记忆 — 每轮对话是独立的，planner 看不到历史消息
日志页面是空的 — system_logs 表里有数据，但后端没提供查询接口，前端也没对接
Dashboard 数据是写死的 — 那些 "12 calls today" 都是静态数字
知识库没有用户隔离 — 所有用户共享同一个知识库，上传没有鉴权
Settings 页面只读 — 不能修改任何配置
没有用户管理 — 没有个人资料页、没有管理员面板
文件上传不支持 DOCX/Excel — 只支持 PDF/TXT/MD
没有分页 — 文档列表、对话列表都是全量加载
没有测试 — 没有 pytest 或前端测试
